✈️ Flight Booking Application

A basic flight booking web application developed using HTML, JavaScript, Node.js, and Express.js. This project demonstrates the core functionality of a flight booking system including displaying flights and booking interactions with a simple frontend and backend setup.

---

🚀 Features

🔹 Front-End

- Displays a list of available flights
- Simple and user-friendly interface
- “Book” button to simulate flight booking
- Instant booking confirmation using alert message
- Uses static flight data for demonstration

---

🔹 Back-End

- Built using Node.js and Express.js
- Basic server setup
- Handles HTTP requests
- Backend ready for further integration with database

---

💻 Technologies Used

- HTML
- JavaScript
- Node.js
- Express.js

---

▶️ Getting Started

🔹 Prerequisites

Make sure you have the following installed:

- Node.js
- npm

---

🔹 Installation

1. Clone or download the repository
2. Open the project folder

---

🔹 Run Backend Server

node server.js

Server will start on:

http://localhost:3000

---

🔹 Run Frontend

- Open "index.html" file in your browser

---
